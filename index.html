Welcome to ICT 8

Build real digital skills you’ll use every day.

This course blends practical file management, typing fluency, web + game development basics, and smart digital citizenship. You’ll learn by doing—with short, focused lessons and plenty of time to practice.

Digital Literacy File & Folder Skills Typing Practice Game Dev (GDevelop) Web Dev (HTML/CSS) OneDrive Workflow

What we’ll be doing

  • Getting set up for success: class expectations, how to use Teams, and where to find + submit work.
  • Core digital skills: creating folders, naming files, saving to specific locations, moving/renaming, downloading & unzipping, and uploading to OneDrive.
  • Typing: a little practice each class to build speed and accuracy.
  • Game Development: build a simple platformer in GDevelop using events and reusable assets.
  • Web Development: create clean, accessible pages with HTML & CSS; organize assets; and publish a basic site.
  • Digital citizenship: passwords, privacy, organization habits, and being a positive community member online.

How to find things

  • Course website: unit pages and lesson instructions live here.
  • Microsoft Teams: announcements, assignments, and submissions.
  • OneDrive: save your work in the correct course folder—you’ll use it across devices.
